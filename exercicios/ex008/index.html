Principais formatções

Negrito ou destaque

Utilizar a tag <b> fará com que o seu título fique e negrio. Porém, é uma tag não semântica, ou seja, possui apenas forma e não contem nenhum significado, pois a letra b é apenas a letra b.

A maneira correta de colocar seu texto em negrito é utilizando a tag <strong>.

Itálico ou Ênfase

Para o uso de itálico usava-se a tag <i>, pois é uma tag não semântica, possui apenas forma.

A maneira correta para se utilizar o itálico é usando a tag <em>.

Texto marcado

Texto grande e pequeno

Para deixar as letras maiores é necessário o uso da tag <big>, porém é uma tag obsoleta. Já para deixar o texto menor usa-se a tag <small>.

Texto deletado

Podemos também realizar um texto deletado, no caso ele poderia ser lido porém sub entende-se que é para ser desconciderado. Desta maneiro: exemplo de texto. Com o uso da tag <del>.
